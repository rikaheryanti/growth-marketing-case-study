# Growth Marketing Case Study

## Overview
This repository documents my setup process and workflow for completing a growth marketing onboarding task. The objective is to demonstrate my ability to set up development tools, use GitHub effectively, and clearly communicate my process.

---

## Tools and Technologies
- Cursor IDE
- Git (Version Control)
- GitHub (Repository Hosting)
- Claude Code extension (Cursor)
- Codex extension (Cursor)

---

## Setup Process

### Environment Setup
- Installed Cursor IDE
- Installed Git on Windows
- Verified Git installation

### GitHub Authentication
- Generated SSH key
- Added SSH key to GitHub
- Verified SSH connection

### Repository Setup
- Cloned the repository
- Opened the project in Cursor

### Extensions Setup
- Installed Claude Code extension
- Installed Codex extension
- Logged into both services

---

## Challenges and Solutions

### Git not recognized
Issue: Terminal could not detect Git  
Solution: Installed Git and restarted the IDE

### SSH authenticity warning
Issue: First-time connection prompt  
Solution: Confirmed connection by trusting the host

### Incorrect clone usage
Issue: Used repository URL as a command  
Solution: Corrected the cloning process using proper Git command

---

## Key Learnings
- Set up a complete development environment from scratch
- Understood Git and version control basics
- Connected GitHub using SSH authentication
- Worked with Cursor IDE and extensions
- Resolved common setup issues

## Reflection
This project helped me understand the full workflow from environment setup to repository management. It also improved my confidence in handling technical setup tasks independently.
